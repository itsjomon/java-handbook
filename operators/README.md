# Operators in Java

Symbols that tell compiler to perform some operation

## Example:

```java
sum = a + b;

// a and b are operands
// + is an operator
```

## Types of Operators

- Arithmetic Operators (Binary/Unary/Ternary)
- Relational Operators
- Logical Operators
- Bitwise Operators
- Assignment Operators
